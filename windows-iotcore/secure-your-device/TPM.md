## <a name="why-tpm"></a><span data-ttu-id="0faf4-110">Warum TPM?</span><span class="sxs-lookup"><span data-stu-id="0faf4-110">Why TPM?</span></span>

<span data-ttu-id="0faf4-115">Computer, auf denen ein TPM verfügen, können auch einen Schlüssel erstellen, der nur nicht umschlossen wurden, aber auch an bestimmte Eigenschaften der Plattform gebunden ist.</span><span class="sxs-lookup"><span data-stu-id="0faf4-115">Computers that incorporate a TPM can also create a key that has not only been wrapped but is also tied to certain platform measurements.</span></span> <span data-ttu-id="0faf4-116">Diese Art von Schlüssel kann nur entschlüsselt werden, wenn die entsprechenden Plattformeigenschaften dieselben Werte aufweisen, die wie beim Erstellen der Schlüssel erstellt wurde.</span><span class="sxs-lookup"><span data-stu-id="0faf4-116">This type of key can only be unwrapped when those platform measurements have the same values that they had when the key was created.</span></span> <span data-ttu-id="0faf4-117">Dieser Prozess heißt **"Versiegeln"** den Schlüssel auf dem TPM.</span><span class="sxs-lookup"><span data-stu-id="0faf4-117">This process is called **"sealing"** the key to the TPM.</span></span> <span data-ttu-id="0faf4-118">Das Entschlüsseln des Schlüssels wird aufgerufen, **"Entsiegeln"**.</span><span class="sxs-lookup"><span data-stu-id="0faf4-118">Decrypting the key is called **"unsealing"**.</span></span> <span data-ttu-id="0faf4-119">Das TPM kann auch versiegeln und Versiegelung von Daten, die außerhalb des TPM.</span><span class="sxs-lookup"><span data-stu-id="0faf4-119">The TPM can also seal and unseal data generated outside of the TPM.</span></span> <span data-ttu-id="0faf4-120">Mit diesem versiegelten Schlüssel und Software wie BitLocker-Laufwerkverschlüsselung können Sie Daten sperren, bis bestimmte Hardware- oder softwarebedingungen erfüllt sind.</span><span class="sxs-lookup"><span data-stu-id="0faf4-120">With this sealed key and software such as BitLocker Drive Encryption, you can lock data until specific hardware or software conditions are met.</span></span>
